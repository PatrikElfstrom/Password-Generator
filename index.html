    Information

    Strong passwords is a must on the Internet today. And luckily there is a lot of good password generators out there.

    But I was not satisfied with the services that existed. I wanted a fast, secure and dynamic password generator.

    So I developed this password generator which is built fully in JavaScript and it generates all passwords locally so no passwords are sent over the Internet. This makes it very fast and secure.
